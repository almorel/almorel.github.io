Encoder les fichiers VOB en h264 / AC3

La commande:

ffmpeg -i concat:VTS_06_1.VOB\|VTS_06_2.VOB\|VTS_06_3.VOB\|VTS_06_4.VOB\|VTS_06_5.VOB\|VTS_06_6.VOB \
    -analyzeduration 100M \
    -probesize 100M \
    -threads 0 \
    -map 0:v \
    -map -i:0x81 \
    -metadata title="TEST" \
    -c:v h264\
    -c:a ac3 \
    TEST.mkv

Les arguments spécifiés dans la ligne de commande sont expliqués dans la documentation de la commande ffmpeg. Dans le tableau ci-dessous, je donne les explications de pourquoi j'utilise les options dans cette ligne de commande:

Option Pourquoi ?
-probesize Pour détecter les pistes audios/vidéos plus loin que par défaut
-analyzeduration idem que -probesize
-analyzeduration idem que -probesize
-map -i:0x81 pour sélectionner la piste 0x81 => pise audio française ??

Rien de spécial dans les autres arguments, je les utilise finalement par défaut.
